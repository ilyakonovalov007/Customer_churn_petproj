# 📉 Предсказание оттока клиентов (Customer Churn Prediction)

## О проекте

Компании теряют деньги каждый раз, когда клиент уходит. Удержать существующего клиента в 5–7 раз дешевле, чем привлечь нового — поэтому задача предсказания оттока имеет прямую бизнес-ценность.

В этом проекте я построил модель, которая по данным о клиенте предсказывает — уйдёт он или останется. Это позволяет бизнесу заранее выявлять группу риска и предпринимать удерживающие меры (скидки, звонки, специальные предложения).

---

## 📦 Данные

**Датасет:** [Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn) (Kaggle)

- 7 043 клиента, 21 признак
- Целевая переменная: `Churn` (Yes / No)
- Дисбаланс классов: 73.5% остались, 26.5% ушли

---

## 🔧 Что сделано

### 1. Разведочный анализ (EDA)
- Выявлен дисбаланс классов → использовали `class_weight='balanced'` в моделях
- `TotalCharges` хранился как строка → преобразован в `float`, пропуски заполнены медианой
- Масштабирование числовых признаков (`tenure`, `MonthlyCharges`, `TotalCharges`) через `StandardScaler`

### 2. Feature Engineering
Добавлены новые признаки для улучшения качества модели:

| Признак | Описание |
|---|---|
| `AverageMonthlyCharges` | Средний исторический чек клиента |
| `LongTermContract` | Флаг долгосрочного контракта |
| `ServicesCount` | Количество подключённых услуг |
| `StreamingCount` | Количество стриминговых сервисов |
| `SecurityCount` | Количество услуг безопасности |

### 3. Модели

| Модель | ROC-AUC | F1 (отток) |
|---|---|---|
| Logistic Regression | 0.8622 | 0.64 |
| Random Forest | 0.8666 | 0.65 |
| CatBoost | ~0.84 | 0.64 |

## 📊 Ключевые выводы

**Что влияет на отток сильнее всего:**

- 🔴 **Fiber optic интернет** — клиенты с этим тарифом уходят чаще (возможно, проблемы с качеством или высокая цена)
- 🔴 **Высокие суммарные платежи (TotalCharges)** — парадоксально, но лояльные долгие клиенты тоже в зоне риска
- 🟢 **Двухлетний контракт** — сильнейший удерживающий фактор
- 🟢 **Длительность в сервисе (tenure)** — чем дольше клиент, тем ниже риск ухода
- 🟢 **Годовой контракт** — также снижает вероятность оттока
- 🟢 **Онлайн-безопасность** — клиенты с защитными сервисами уходят реже

**Практический вывод для бизнеса:**  
Клиент на помесячном контракте + Fiber optic + короткий срок жизни = высокий риск оттока. Именно на эту группу стоит направить удерживающие кампании в первую очередь.
